# 你好，世界！

这是我用 **Hugo** 和 **GitHub Pages** 搭建的个人博客，完全免费，无需服务器。

## 为什么要写博客？

- 整理和沉淀自己的知识
- 记录学习过程中的思考
- 与更多人分享有价值的内容

## 技术栈

| 工具 | 作用 |
|------|------|
| Hugo | 静态网站生成器 |
| PaperMod | 博客主题 |
| GitHub Pages | 免费托管 |
| GitHub Actions | 自动部署 |
